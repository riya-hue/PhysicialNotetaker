A simple AI tool that automatically creates medical notes from doctor-patient conversations.

## **What It Does**
- **Extracts Medical Details** - Pulls out symptoms, diagnosis, treatment, and prognosis from conversations
- **Analyzes Patient Mood** - Determines if the patient is anxious, reassured, or neutral  
- **Creates Structured Notes** - Organizes information into standard medical SOAP format

## **How It Works**
1. Takes a doctor-patient conversation transcript
2. Uses keyword matching to find medical information
3. Analyzes patient statements for emotional tone
4. Structures everything into a clean medical report

## **Setup Instructions**
1. Install Python (version 3.8 or higher recommended)
2. Install Jupyter Notebook (if using locally) or use Google Colab
3. Run all cells in the notebook or execute the Python script
