A simple AI tool that automatically creates medical notes from doctor-patient conversations.

What It Does
(i)Extracts Medical Details - Pulls out symptoms, diagnosis, treatment, and prognosis from conversations

(ii)Analyzes Patient Mood - Determines if the patient is anxious, reassured, or neutral

(iii)Creates Structured Notes - Organizes information into standard medical SOAP format

How It Works
a)Takes a doctor-patient conversation transcript

b)Uses keyword matching to find medical information

c)Analyzes patient statements for emotional tone

Structures everything into a clean medical report

Setup Instructions
1)Install Python (version 3.8 or higher recommended)

2)Install Jupyter Notebook (if using locally) or use Google Colab

Run all cells in the notebook or execute the Python script
